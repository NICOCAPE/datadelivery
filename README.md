# DATA DELIVERY

---

## 📝 Descrizione

> **Anno 2089. La città di Neo-Lynx è sotto attacco hacker.**
> Energia, trasporti e comunicazioni sono in crisi: il flusso dei dati è corrotto e il collasso totale è imminente.

> Nei panni di un agente speciale, il giocatore deve raccogliere i pacchetti dati dispersi nella rete e consegnarli intatti al server centrale, evitando i bug che infestano il sistema.
> Il database accetta solo dati puri: se la percentuale di corruzione supera il 10%, la missione fallisce.

> Tre livelli di difficoltà — **Facile**, **Media**, **Difficile** — mettono alla prova riflessi e strategia, con bug sempre più veloci e numerosi.
> Solo un trasferimento pulito può salvare la città.
